☕ Cozy Data Quality Dashboard

Data Drift Trust Platform

This project monitors CRM, Billing, and Analytics data for missing records, duplicate entries, value mismatches, and drift over time.

📌 Missing Records

Checks whether expected data is absent from any system.

🔁 Duplicates

Identifies repeated records that may distort analysis.

⚠️ Mismatches

Highlights differences in values across data sources.

⭐ Trust Score

Provides a summarized quality score for the dataset.
